How to Get Help

Student Support for eLC

If you need assistance with eLC, you have a number of options to get help using the system.

Contact the EITS Help Desk: helpdesk@uga.edu or 706.542.3106.
Business hours are 8am - 10pm (Mon-Thurs), 8am-6pm (Fri), and 1pm-7pm (Sat-Sun)
  
Go the University System of Georgia Desire2Learn Online Help Center at http://D2Lhelp.view.usg.edu. From this site, you can search the Knowledge Base, browse popular articles, or click the link for live support (24x7x365) in the scrolling news area.

Use the course discussion boards

There are two discussion boards where you can ask for help. The Ask a Classmate - Technical and Ask a Classmate - Assignment boards are areas that you should head to first to see if others are having issues with the same problems. It's also a great place to get a quick answer when needed.

My instructor is using Online Rooms. Where can I get help?
