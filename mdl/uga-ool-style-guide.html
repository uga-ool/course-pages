UGA Online eLC Content Template Style Guide

Paragraph Text

Default paragraph text uses the Merriweather Sans font with Tahoma and sans-serif as fallbacks. Default font weight is 300, and font size is set to 1em, which means body/paragraph fonts will be set to the default size of the user's browser.

Special Text Classes

There is a "text-center" class that can be applied to text elements to center text horizontally on the screen.

There is a "heavy" class that can be added to text elements to change the font-weight to 400. For example:

This text will be <span class="heavy">bold</span>

This will appear as:

This text will be bold

Headings

Default heading text uses the Merriwheather font with serif font as a fallback. All unclassed headings have a base top & bottom margin of 5px and a line height of 2em.

This is an h1.

H1 should not be used in the body of our pages because it is used at the top left for the course name

This is an h2.

H2 should serve as top level headings in the body of our pages. The base size for h2 is 2em

This is an h3.

The base size for h3 is 1.6em.

This is an h4.

The base size for h4 is 1.3em.

This is an h5.

The base size for h5 is 1.1em.

This is an h6.

The base size for an h6 is 1em

Special heading classes

There is a special h3 class for subtitles that adds a negative margin to the top of the heading to move it closer to the main title. To use it, add a class of "subtitle" to your h3, like so:

<h3 class="subtitle">Subtitle Goes Here</h3>

This can be combined with the text-center class to produce a centered page title and subtitle:

<h2 class="text-center">Page Title</h2>
<h3 class="subtitle text-center">Page Subtitle</h3>

Which will produce:

Page Title

Page Subtitle

Lists

In order to produce an unordered list or ordered list with no bullet points/numbers and with no extra margin or padding, apply a class of "plain" to your <ul> or <ol>. For example:

<ul class="plain">
  <li>List Item 1</li>
  <li>List Item 2</li>
  <li>List Item 3</li>
  <li>List Item 4</li>
</ul>

Will produce:

  • List Item 1
  • List Item 2
  • List Item 3
  • List Item 4

If you want a bit more spacing between your line items, apply a class of "roomy" to your list. For example:

<ul class="roomy">
  <li>List Item 1</li>
  <li>List Item 2</li>
  <li>List Item 3</li>
  <li>List Item 4</li>
</ul>

Will produce:

  • List Item 1
  • List Item 2
  • List Item 3
  • List Item 4

Tables

Properly formatted HTML tables will take the following form:

<table>
  <tr>
    <th>Table Header</th>
    <th>Table Header</th>
  </tr>
  <tr>
    <td>Table Data</td>
    <td>Table Data</td>
  </tr>
  <tr>
    <td>Table Data</td>
    <td>Table Data</td>
  </tr>
  <tr>
    <td>Table Data</td>
    <td>Table Data</td>
  </tr>
</table>

Which will render:

Table Header Table Header
Table Data Table Data
Table Data Table Data
Table Data Table Data

There are a variety of classes you can add to the table markup to customize its appearance. They are detailed below.

Table Width

The table element can have a class of "full-table" or "half-table" applied to it in order to make it span 100% or 50% of its container, respectively.

Column Width

The th elements can have a variety of classes applied to them in order to influence their width. Those classes are:

  • table-column-50
  • table-column-75
  • table-column-100
  • table-column-125
  • table-column-150
  • table-column-200
  • table-column-250
  • table-column-400
  • table-column-500

The number in the class is equivalent to the width the column will be set to in pixels. Please note that if you use "full-width" or "half-width" in conjunction with these classes, the columns will scale to accomodate the actual size of the table. Therefore, if your 100% table is 800px wide, but the total width of your specified columns only equal 600px, then the columns will scale in porportion to your requested widths.

Cell Background Color

By default, the th elements will have a background color of #666666 and a font color of #ffffff. Additionally, you can give table rows a background color of #f4f4f4 by applying the class "table-row-grey" to the tr element. Applying this class to every other row can provide a pleasing visual effect, as the example below will illustrate.

Table Alignment

Tables can be centered horizontally on the screen by applying a "center" class to the table element. This will also result in the text being centered in each cell of the table.

Table Example

Here is an example of a table using some of the options described above.

<table class="half-table center">
  <tr>
    <th class="table-column-75">Table Header</th>
    <th class="table-column-150">Table Header</th>
  </tr>
  <tr>
    <td>Table Data</td>
    <td>Table Data</td>
  </tr>
  <tr>
    <td class="table-row-grey">Table Data</td>
    <td class="table-row-grey">Table Data</td>
  </tr>
  <tr>
    <td>Table Data</td>
    <td>Table Data</td>
  </tr>
</table>

Which will produce:

Table Header Table Header
Table Data Table Data
Table Data Table Data
Table Data Table Data

Images

Images can be placed in containers that serve to set the image apart from the text content, provide a space for an image caption and source link, and allow for the wrapping of text around the image. The following table outlines the available image container sizes and the required image size for each container. You can use the Kaltura Thumbnail API in order to render images to the required size. Please note that these images sizes are responsive, so they will shrink at smaller screen resolutions.

All dimensions in pixels.

Image Container Size Required Image Size Class Name
500w x 500h 500w x 400h card-square-500
400w x 400h 400w x 300h card-square-400
300w x 300h 300w x 200h card-square-300
512w x 300h 512w x 200h card-wide-512
800w x 450h 800w x 350h card-wide-800

Image Alignment

Images can be aligned left, right, or center by applying one of the following classes to the outermost div of the image code:

  • card-left
  • card-right
  • card-center

Here is example code for a left-aligned, 500 by 500 image area

<div class="card-left card-square-500 mdl-card mdl-shadow--2dp">
  <div class="mdl-card__title mdl-card--expand">
    <img src="Image Source URL Goes Here">
  </div>
  <div class="mdl-card__supporting-text">
    Image Caption Goes Here.
  </div>
  <div class="mdl-card__actions mdl-card--border">
    <a class="mdl-button mdl-button--colored mdl-js-button mdl-js-ripple-effect"
    href="Image Source URL Here"
    target="_blank">
    View Image Source
    </a>
  </div>
</div>

Which will render:

Image Caption Goes Here.
View Image Source

Videos

Videos can be placed in card containers similar to images. Videos embedded using the LMS embed codes have a dimension of 585w X 420h, which appears to be unchangeable. When embedding into the LMS, you can use the following embed code, replacing each value that contains the "XXXXXXXX" placeholder:

<iframe>id="VideoCaptureIframe" src="/d2l/im/kaltura/MediaPlayer.d2l?ou={OrgUnitId}&entryId=XXXXXXXX&title=XXXXXXXXXX
    &description=&mediaType=1" width="540" height="304" scrolling="no" frameborder="0" title="XXXXXXXX"
    allowfullscreen="true" webkitallowfullscreen="true" mozallowfullscreen="true">Kaltura Video Player</iframe>

In order to place a video inside a card, you must wrap the iframe in a div with the class "video-card". There are a few optional styles to customize the card, detailed in the table below:

Div Class Effect on video card
video-card Creates video card element
video-left Floats video card to the left, causes text to wrap around the right side of the video player. Takes effect at 920w
video-right Floats video card to the right, causes text to wrap around the left side of the video player. Takes effect at 920w
video-dark Changes the card color to dark gray & lightens the text color
video-card-wide Widens the card and associated description area to take up the entire width of the content window. Must be used with a special wide player (not the standard D2L player).

In order to add the title, horizontal rule, and description to the video card, you should add a span with class "video-title", a horizontal rule element, and a span with class "video description" after the iframe inside the "video-card" div. The full video card HTML will look like this:

<div class="video-card video-left">
    <iframe id="VideoCaptureIframe" src="/d2l/im/kaltura/MediaPlayer.d2l?ou={OrgUnitId}&entryId=XXXXXXXX&title=XXXXXXXXXX&description=&mediaType=1"
        width="585" height="420" scrolling="no" frameborder="0" title="XXXXXXXX" allowfullscreen="true"
        webkitallowfullscreen="true" mozallowfullscreen="true">Kaltura Video Player</iframe>
    <span class="video-title">Video Title</span>
    <hr>
    <span class="video-description">Video description.</span>
</div>

Which will render as:

Video Title
Video description.
